# Sons de notification (matin, midi, soir)

Place dans ce dossier **3 fichiers WAV** pour avoir un son différent à chaque créneau :

- **morning.wav** — notification du matin (ex. 8h30)
- **noon.wav** — notification de midi (ex. 12h30)
- **evening.wav** — notification du soir (ex. 21h)

Recommandations :
- Format : **WAV** (recommandé par Expo), courte durée (quelques secondes).
- Si les fichiers sont absents, l’app utilisera le son par défaut du téléphone.

Tu peux utiliser des sons libres de droit (ex. Freesound.org) ou enregistrer les tiens.
